# Charte Graphique — Sakura Ramen 🌸

Ce document présente l'identité visuelle et les maquettes interactives du projet Sakura Ramen, capturées directement sur le site en production : https://sakura-ramen.vercel.app

## 1. Identité Visuelle
L'univers de Sakura Ramen s'inspire de l'esthétique **Ryoko**, mêlant tradition japonaise et modernité minimaliste.

### Palette de Couleurs
- **Fond principal** : `#1A0A1F` (Deep Violet Rose)
- **Accents** : `#FF69B4` (Sakura Pink)
- **Texte** : `#FFF5F7` (Creamy White)

### Typographie
- **Titres** : `Special Elite` (Authenticité typewriter)
- **Corps** : `Inter` (Lisibilité moderne)

## 5. Ergonomie Mobile (Optimisée)
Le design s'adapte aux petits écrans pour une utilisation fluide à une main :
- **Navigation Drawer** : Un menu "burger" escamotable libère l'espace visuel.
- **Tableaux Responsifs** : Dans l'admin, les données complexes sont converties en "cartes" empilées.
- **Touch Targets** : Zones de clic de 48px minimum pour un confort tactile optimal.
